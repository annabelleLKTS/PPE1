Dans le cadre du cours Projet Encadré 1 du Master TAL, nous avons un mini-projet a réaliser.

Pour cela nous devons :

  • Constituer un groupe de 3 étudiants
  • Choisir 3 langues différentes. Exemple : français - arabe - tamoul
  • Choisir un mot différent par langue mais au sens proche
  • Utiliser un script et les fonctions abordés tel que : curl, lynx, echo, while
  • Constituer un corpus de 50 liens par langues, pour chage page internet où apparait le mot choisi dans un texte
  • Présenter le corpus sous forme de tableau de résultat
  • Créer une page internet git-hub avec HTMl et du style CSS, afin d'y présenter notre mini-projet
  • Tout cela sous linux
